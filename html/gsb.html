Projet GSB - Application de Gestion de Frais

Simplifiez la gestion des frais professionnels avec une application intuitive et performante.

Introduction

Le projet GSB (Galaxy Swiss Bourdin) consiste à développer une application de gestion de frais professionnels pour les collaborateurs de l’entreprise. Cette solution permet de simplifier le suivi, la déclaration, et la validation des frais engagés par les employés, tout en améliorant la transparence et l’efficacité administrative.

Fonctionnalités Principales

Déclaration des frais

Les employés peuvent facilement soumettre leurs frais avec des justificatifs directement depuis l'application.

Validation rapide

Les managers disposent d’une interface intuitive pour examiner et approuver les frais soumis.

Suivi en temps réel

Consultez l’état des demandes et les dépenses en cours avec des tableaux de bord dynamiques.

Rapports détaillés

Générez des rapports pour une meilleure analyse des coûts et une prise de décision éclairée.

Technologies Utilisées

Frontend

HTML5, CSS3, JavaScript (avec frameworks comme React ou Vue.js).

Backend

PHP pour la gestion des APIs et la logique serveur.

Base de Données

MySQL pour stocker et gérer les données de manière sécurisée.
